Application Name
Keepr
Overview
Keepr is a social network where users can share and explore their interests by posting images or videos to their personal or others' collections. This project showcases the ability to build full-stack applications using Vue.js, DotNet WebApi, Auth0, Dapper ORM, and MySQL database. The frontend uses Vue-Router to manage the DOM, while the server-side implements the Repository pattern to communicate with the database.
